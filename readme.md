# ajmalrazzaq07.github.io

This is a personal website belonging to Ajmal Razzaq. The website serves as a portfolio to showcase his skills, projects, and interests.

## Site Structure

The website consists of several pages, including:

- Home: A brief introduction to Ajmal and his skills.
- About: A more detailed description of Ajmal's background, education, and work experience.
- Projects: A showcase of Ajmal's software development projects.
- Blog: A collection of blog posts written by Ajmal on various topics related to software development and technology.
- Contact: A page where visitors can get in touch with Ajmal.

## Technologies Used

The website is built using the following technologies:

- HTML
- CSS
- JavaScript
- Bootstrap framework

## Usage

The website is publicly accessible at https://ajmalrazzaq07.github.io/. Visitors can browse the various pages to learn more about Ajmal, his skills, and his projects. They can also contact him using the information provided on the Contact page.
